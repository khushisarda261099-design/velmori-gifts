## Overview

Velmori Gifts is a responsive, elegantly designed landing page for a premium gift shop. The project features a clean, modern aesthetic with custom animations and interactive elements.

## Features

- **Responsive Design**: Fully optimized for desktop and mobile devices
- **Modern UI**: Clean typography with custom fonts (Cormorant Garamond, Lato, Dancing Script)
- **Interactive Elements**:
  - Animated hero badge with floating effect
  - Hover animations on product cards
  - Marquee strip for featured offerings
  - Sparkle effects and decorative elements
- **Sections**:
  - Hero/navigational header with WhatsApp CTA
  - About section with statistics
  - Product showcase grid
  - Testimonials carousel
  - Contact information card

## Getting Started

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- No server required - static HTML file

### Installation

1. Clone the repository:
```bash
git clone https://github.com/yourusername/velmori-gifts.git
```

2. Open `index.html` directly in your browser or serve via any static server.

## Project Structure

```
velmori-gifts/
├── index.html      # Main HTML file with embedded styles
└── README.md       # This documentation
```

## Customization

To customize the website:

1. **Colors**: Modify CSS variables in the `:root` selector at the top of `index.html`
2. **Fonts**: Update Google Fonts link in the `<head>` section
3. **Content**: Edit product cards, testimonials, and text content within `index.html`
